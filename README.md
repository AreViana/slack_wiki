# Welcome
This project was created with the purpose of sharing the knowledge gathered in the study group of girls from the Latin American slackline community.

## ⚡ Project Stack
- Vitepress
- Typescript

## 📂 Project Structure
```
├─ docs/              # Project root for VitePress
│  ├─ .vitepress/     # Config directory
│  │  └─ config.mts   # Main configuration file
│  ├─ public/         # Public directory
│  │  └─ assets/      # Assets directory
│  ├─ index.md        # Home page
│  └─ pages.md        # Main site pages
└─ package.json
```

## 🛠️ Available Scripts
- `npm run docs:dev`: Start a local development server with hot-reload.
- `npm run docs:build`: Build the site for production.
- `npm run docs:preview`: Locally preview the production build.
